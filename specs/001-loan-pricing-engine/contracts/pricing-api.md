# API Contract: Loan Pricing Engine

**Branch**: `001-loan-pricing-engine` | **Date**: 2026-05-30
**Base URL**: `/api/v1`
**Format**: JSON request/response; `Content-Type: application/json`

---

## POST /price/fixed

Calculate monthly payment and APR for a fixed-rate home loan.

### Request Body

```json
{
  "scenario_id": "optional-uuid-string",
  "home_price": "400000.00",
  "down_payment": "80000.00",
  "term_years": 30,
  "annual_rate": "6.50",
  "fee_schedule": {
    "fees": [
      { "name": "Origination fee", "amount": "1500.00" }
    ]
  }
}
```

**Field notes**:
- `scenario_id`: optional; generated by server if omitted.
- `home_price`, `down_payment`, `annual_rate`: decimal strings (not floats) to preserve precision.
- `down_payment`: may be a dollar string (`"80000.00"`) or percentage string (`"20%"`).
- `fee_schedule`: optional; omit or pass `{"fees": []}` for no fees.
- `term_years`: integer; common values 10, 15, 20, 30.

### Success Response — 200 OK

```json
{
  "scenario_id": "a1b2c3d4-...",
  "loan_type": "fixed",
  "loan_amount": "320000.00",
  "monthly_payment": "2022.65",
  "apr": "6.612",
  "total_interest": "408154.00",
  "rate_type_label": "Fixed 30-year",
  "parameters_summary": {
    "home_price": "400000.00",
    "down_payment": "80000.00",
    "term_years": 30,
    "annual_rate": "6.50"
  }
}
```

### Validation Error — 422 Unprocessable Entity

```json
{
  "detail": [
    {
      "scenario_id": null,
      "code": "INVALID_RATE",
      "message": "annual_rate must be between 0 and 30",
      "fields": ["annual_rate"]
    }
  ]
}
```

## POST /price/arm

Calculate initial monthly payment and APR for an adjustable-rate home loan.

### Request Body

```json
{
  "scenario_id": "optional-uuid-string",
  "home_price": "350000.00",
  "down_payment": "70000.00",
  "term_years": 30,
  "initial_rate": "5.75",
  "arm_params": {
    "fixed_period_years": 5,
    "adjustment_period_years": 1,
    "initial_cap": "2.00",
    "periodic_cap": "2.00",
    "lifetime_cap": "5.00"
  },
  "fee_schedule": {
    "fees": []
  }
}
```

### Success Response — 200 OK

```json
{
  "scenario_id": "b2c3d4e5-...",
  "loan_type": "arm",
  "loan_amount": "280000.00",
  "monthly_payment": "1634.17",
  "apr": "5.893",
  "total_interest": "308301.20",
  "rate_type_label": "5/1 ARM 30-year",
  "parameters_summary": {
    "home_price": "350000.00",
    "down_payment": "70000.00",
    "term_years": 30,
    "initial_rate": "5.75",
    "arm_params": {
      "fixed_period_years": 5,
      "adjustment_period_years": 1,
      "initial_cap": "2.00",
      "periodic_cap": "2.00",
      "lifetime_cap": "5.00"
    }
  }
}
```

**Note**: `apr` for ARM is disclosed using the initial-rate-held-constant assumption
per Regulation Z Appendix J.

### Validation Error — 422 Unprocessable Entity

Same structure as `/price/fixed`. Additional ARM-specific codes:

| Code | Meaning |
|------|---------|
| `PERIODIC_CAP_EXCEEDS_LIFETIME` | `periodic_cap` > `lifetime_cap` |
| `FIXED_PERIOD_EXCEEDS_TERM` | `fixed_period_years` ≥ `term_years` |
| `MAX_RATE_EXCEEDS_CEILING` | `initial_rate` + `lifetime_cap` > 30 |

## POST /price/compare

Price and compare up to 5 loan scenarios in a single request.

### Request Body

```json
{
  "comparison_id": "optional-uuid-string",
  "scenarios": [
    {
      "scenario_id": "s1",
      "loan_type": "fixed",
      "home_price": "400000.00",
      "down_payment": "80000.00",
      "term_years": 30,
      "annual_rate": "6.50"
    },
    {
      "scenario_id": "s2",
      "loan_type": "fixed",
      "home_price": "400000.00",
      "down_payment": "80000.00",
      "term_years": 15,
      "annual_rate": "6.00"
    }
  ]
}
```

**Field notes**:
- `scenarios`: 1–5 items; validated per-item.
- Each item follows the same schema as the individual `/price/fixed` or `/price/arm`
  body, with the addition of a `loan_type` discriminator field.

### Success Response — 200 OK

```json
{
  "comparison_id": "c3d4e5f6-...",
  "results": [
    {
      "scenario_id": "s1",
      "loan_type": "fixed",
      "loan_amount": "320000.00",
      "monthly_payment": "2022.65",
      "apr": "6.50",
      "total_interest": "408154.00",
      "rate_type_label": "Fixed 30-year"
    },
    {
      "scenario_id": "s2",
      "loan_type": "fixed",
      "loan_amount": "320000.00",
      "monthly_payment": "2702.39",
      "apr": "6.00",
      "total_interest": "166430.20",
      "rate_type_label": "Fixed 15-year"
    }
  ]
}
```

### Partial Success — 200 OK (mixed valid/invalid)

When one or more scenarios are invalid, results for valid scenarios are returned
alongside error details for invalid ones. HTTP status is still 200; callers MUST
inspect each result item's type.

```json
{
  "comparison_id": "c3d4e5f6-...",
  "results": [
    {
      "scenario_id": "s1",
      "loan_type": "fixed",
      "monthly_payment": "2022.65",
      "apr": "6.50",
      "..."
    },
    {
      "scenario_id": "s2",
      "error": true,
      "code": "INVALID_RATE",
      "message": "annual_rate must be between 0 and 30",
      "fields": ["annual_rate"]
    }
  ]
}
```

### Validation Error — 422 Unprocessable Entity

Returned only when the top-level request structure is invalid (e.g., `scenarios` is
empty or contains more than 5 items).

```json
{
  "detail": [
    {
      "code": "SCENARIO_LIMIT_EXCEEDED",
      "message": "A comparison request may contain at most 5 scenarios",
      "fields": ["scenarios"]
    }
  ]
}
```

## GET /health

Liveness check. Returns 200 with `{"status": "ok"}`. No auth required.

---

## Error Codes Reference

| Code | HTTP | Meaning |
|------|------|---------|
| `INVALID_RATE` | 422 | Rate out of [0, 30] range |
| `INVALID_TERM` | 422 | Term ≤ 0 or > 50 years |
| `DOWN_PAYMENT_EXCEEDS_PRICE` | 422 | Down payment ≥ home price |
| `ZERO_LOAN_AMOUNT` | 422 | Derived loan amount is 0 |
| `INVALID_CAP` | 422 | ARM cap value out of range |
| `PERIODIC_CAP_EXCEEDS_LIFETIME` | 422 | periodic_cap > lifetime_cap |
| `FIXED_PERIOD_EXCEEDS_TERM` | 422 | fixed_period_years ≥ term_years |
| `MAX_RATE_EXCEEDS_CEILING` | 422 | initial_rate + lifetime_cap > 30 |
| `SCENARIO_LIMIT_EXCEEDED` | 422 | More than 5 scenarios in compare |
| `APR_CONVERGENCE_FAILURE` | 500 | APR solver did not converge (rare) |
